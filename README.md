# MongoChat

Simple chat app that uses MongoDB and Socket.io

<li> chat application with Node.js, MongoDB and Socket.io. </li>
<li> using Bootstrap 4 as well as the client side JavaScript </li>


### Version
1.0.0

## Install Dependencies
```bash
npm install 
```

## Run Server
```bash
npm start
```

## Run App
Open index.html
